# SVG-encoder for Quasar app icons

## Technology stack
- **UI**: [`Quasar`](https://quasar.dev/)
- **Lint**: [`eslint`](https://eslint.org/), [`prettier`](https://prettier.io/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
